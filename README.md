# Спецификация модулей библиотеки статистической аналитики

## Общие положения
- Все функции, принимающие итераторы, должны корректно работать с любым итерируемым объектом (списки, кортежи, генераторы).
- Если функция требует нескольких проходов по данным, она должна либо загружать данные в память (с предупреждением), либо реализовывать однопроходный алгоритм.
- При пустых входных данных выбрасывать исключение `StatisticsError` (можно определить свой класс или использовать `ValueError`).
- Для математических операций использовать модуль `math`.
- Декораторы должны сохранять имя и документацию обёрнутой функции (`functools.wraps`).

## Модуль `core.py`
Базовые статистические функции.

### `mean(data)`
- **Параметры**: `data` – итератор чисел (int/float).
- **Возвращает**: `float` – среднее арифметическое.
- **Исключения**: `StatisticsError` если `data` пуст.
- **Примечание**: вычисляется за один проход.

### `median(data)`
- **Параметры**: `data` – итератор чисел.
- **Возвращает**: `float` – медиана.
- **Исключения**: `StatisticsError` если `data` пуст.
- **Примечание**: требует сортировки, поэтому все данные загружаются в память. Для чётного количества – среднее двух центральных элементов.

### `mode(data)`
- **Параметры**: `data` – итератор чисел.
- **Возвращает**: `float` – значение моды (наиболее часто встречающееся). Если несколько мод – возвращается первая встретившаяся (или можно возвращать список, но для простоты – первая).
- **Исключения**: `StatisticsError` если `data` пуст.
- **Примечание**: использует `collections.Counter`.

### `variance(data, ddof=0)`
- **Параметры**:
  - `data` – итератор чисел.
  - `ddof` – int, delta degrees of freedom. 0 – смещённая дисперсия (деление на n), 1 – несмещённая (деление на n-1).
- **Возвращает**: `float` – дисперсия.
- **Исключения**: `StatisticsError` если `data` пуст или если `ddof >= len(data)`.
- **Примечание**: реализуется онлайн-алгоритмом (один проход) для эффективности.

### `std(data, ddof=0)`
- **Параметры**: аналогично `variance`.
- **Возвращает**: `float` – стандартное отклонение (квадратный корень из дисперсии).

## Модуль `correlation.py`
Корреляционный анализ.

### `covariance(x, y, ddof=1)`
- **Параметры**:
  - `x`, `y` – итераторы чисел одинаковой длины.
  - `ddof` – int, обычно 1 для несмещённой ковариации.
- **Возвращает**: `float` – ковариация.
- **Исключения**: `ValueError` если длины не совпадают или последовательности пусты.
- **Примечание**: вычисляется за один проход с накоплением сумм.

### `pearson_correlation(x, y)`
- **Параметры**: `x`, `y` – итераторы чисел одинаковой длины.
- **Возвращает**: `float` – коэффициент корреляции Пирсона (в диапазоне [-1, 1]).
- **Исключения**: `ValueError` если длины не совпадают или стандартное отклонение одной из выборок равно 0.
- **Примечание**: использует `covariance` и `std`.

## Модуль `regression.py`
Простая линейная регрессия.

### `linear_regression(x, y)`
- **Параметры**: `x`, `y` – итераторы чисел одинаковой длины.
- **Возвращает**: словарь (или `namedtuple`) с полями:
  - `slope` – коэффициент наклона.
  - `intercept` – свободный член.
  - `r_squared` – коэффициент детерминации R².
  - `mse` – средняя квадратичная ошибка.
  - `predictions` – список предсказанных значений для входных `x` (если `x` был итератором, преобразуется в список).
  - `residuals` – список остатков (разность между фактическими и предсказанными).
- **Исключения**: `ValueError` если длины не совпадают или данные пусты.
- **Примечание**: вычисляется методом наименьших квадратов.

### `predict(model, new_x)`
- **Параметры**:
  - `model` – словарь, возвращённый `linear_regression`.
  - `new_x` – число или итератор чисел.
- **Возвращает**: если `new_x` число – float, предсказанное значение; если итератор – генератор предсказанных значений.
- **Исключения**: `TypeError` если модель некорректна.

## Модуль `outliers.py`
Обнаружение и обработка выбросов.

### `detect_outliers_iqr(data, k=1.5)`
- **Параметры**:
  - `data` – итератор чисел.
  - `k` – множитель для межквартильного размаха (обычно 1.5).
- **Возвращает**: список индексов элементов, являющихся выбросами.
- **Исключения**: `StatisticsError` если данных недостаточно для вычисления квартилей.
- **Примечание**: все данные загружаются в память для вычисления квантилей.

### `remove_outliers(data, method='iqr', **kwargs)`
- **Параметры**:
  - `data` – итератор чисел.
  - `method` – строка, пока только `'iqr'`.
  - `**kwargs` – дополнительные аргументы для метода (например, `k`).
- **Возвращает**: генератор, выдающий элементы `data`, не являющиеся выбросами.
- **Примечание**: для метода `iqr` сначала вычисляются границы (требуется полный проход), затем данные фильтруются. Использует `detect_outliers_iqr` для получения индексов или границ. Если `data` – генератор, он будет преобразован в список для первого прохода, но второй проход будет выдавать значения лениво.

## Модуль `streaming.py`
Потоковые вычисления и генераторы.

### `read_numbers_from_file(file_path)`
- **Параметры**: `file_path` – путь к текстовому файлу.
- **Возвращает**: генератор чисел (float), извлекаемых из файла. Каждая строка может содержать одно или несколько чисел, разделённых пробелами, запятыми или табуляцией. Пустые строки игнорируются.
- **Примечание**: файл открывается в контекстном менеджере, генератор читает строки по мере необходимости.

### `sliding_window(iterable, window_size)`
- **Параметры**:
  - `iterable` – любой итератор.
  - `window_size` – положительное целое число.
- **Возвращает**: генератор кортежей, каждый размером `window_size`, представляющих последовательные окна из `iterable`. Если длина итератора меньше `window_size`, генератор пуст.
- **Примечание**: использует `collections.deque` для эффективности.

### `streaming_mean(iterable)`
- **Параметры**: `iterable` – итератор чисел.
- **Возвращает**: генератор, выдающий текущее среднее значение после каждого нового элемента (начиная с первого).
- **Примечание**: реализует онлайн-алгоритм: хранит `count` и `total`.

### `streaming_variance(iterable, ddof=0)`
- **Параметры**:
  - `iterable` – итератор чисел.
  - `ddof` – int, аналогично `variance`.
- **Возвращает**: генератор, выдающий текущую дисперсию после каждого нового элемента. Для первых двух элементов дисперсия может быть `NaN` или 0 (по договорённости).
- **Примечание**: использует алгоритм Велфорда (Welford's online algorithm) для численной устойчивости.

### `streaming_pearson(x_iter, y_iter)`
- **Параметры**: `x_iter`, `y_iter` – итераторы чисел, попарно синхронизированные.
- **Возвращает**: генератор, выдающий текущий коэффициент корреляции Пирсона после каждой новой пары. До получения двух пар возвращает `None` или 0.
- **Примечание**: накапливает суммы `n`, `sum_x`, `sum_y`, `sum_xy`, `sum_x2`, `sum_y2`. Вычисление корреляции по формулам.

## Модуль `decorators.py`
Декораторы для обёртывания функций.

### `@timer`
- **Описание**: выводит в консоль время выполнения декорируемой функции в секундах (с точностью до миллисекунд).
- **Использование**: применить к любой функции.

### `@logger`
- **Описание**: логирует вызов функции: имя, переданные аргументы (позиционные и именованные) и возвращаемое значение. Использует `print` или стандартный модуль `logging`.
- **Использование**: применить к функции.

### `@validate_numeric`
- **Описание**: проверяет, что все аргументы-коллекции (списки, кортежи) содержат только числа (`int` или `float`). Если встречается нечисловой элемент, выбрасывает `TypeError`. Генераторы не проверяются (предполагается, что пользователь гарантирует корректность).
- **Использование**: применить к функциям, которые ожидают числовые данные (например, `mean`, `variance`).

### `@memoize(maxsize=None)`
- **Описание**: кеширует результаты функции для заданных аргументов. Если `maxsize` задан, кеш ограничен указанным числом элементов (LRU-стратегия). Для реализации LRU можно использовать `collections.OrderedDict`.
- **Использование**: применить к вычислительно затратным функциям с хешируемыми аргументами.

## Модуль `utils.py`
Вспомогательные функции.

### `ensure_list(iterable)`
- **Параметры**: `iterable` – любой итератор.
- **Возвращает**: список, содержащий все элементы `iterable`.
- **Примечание**: если `iterable` является генератором, выводит предупреждение (`warnings.warn`), так как это может привести к загрузке всех данных в память.

### `is_numeric(value)`
- **Параметры**: `value` – любой объект.
- **Возвращает**: `bool` – True, если значение является `int` или `float` (не `complex` и не `bool`). Для `bool` возвращает False, так как `bool` является подклассом `int`, но обычно не рассматривается как число в статистике.
